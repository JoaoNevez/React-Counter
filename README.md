# React Counter

Um projeto simples em **React + TypeScript** feito com Vite.

## 🚀 Rodando o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Execute em modo dev:
   ```bash
   npm run dev
   ```

3. Abra no navegador:
   [http://localhost:5173](http://localhost:5173)

## 📦 Build
```bash
npm run build
npm run preview
```
